# frontend/assets/images

Use this folder for static visual assets used across pages.

## Suggested grouping

- Branding: logos
- Hero/Banner visuals
- Section illustrations
- Empty-state visuals

## File naming examples

- `brand-logo-light.png`
- `doctor-dashboard-banner.webp`
- `empty-no-results.svg`

## Quality guidelines

- Prefer WebP/SVG where possible.
- Compress large PNG/JPG before commit.
- Keep one source of truth for each image (no renamed duplicates).

Icons belong in `assets/icons/`, not here.
